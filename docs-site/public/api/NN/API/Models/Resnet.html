    ResNet Model Helpers (API) #

    This module provides a ResNet-style classifier constructor used by runnable examples.

    The architecture is small enough to keep the Lean shape arguments readable while still following the standard ResNet pattern:

    • a 3×3 conv stem + BatchNorm + ReLU
    • three resnetBasicBlocks (one downsampling)
    • global average pooling
    • linear classifier head

    Our goal is to keep the runnable gondlin resnet example focused on data loading, optimizer choice, and reporting. The positivity proofs below are Lean's way of recording that pooling axes are nonempty; callers should normally just use nn.models.resnet cfg.
