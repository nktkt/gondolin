    Gondlin-executable model: FNO1D #

    This is the Models-namespaced alias for Gondlin's 1D Fourier Neural Operator implementations.

    The actual implementation lives in NN.Runtime.Autograd.Gondlin.Fno1d under Runtime.Autograd.Gondlin.NN.FNO1D. We keep the executable architecture definitions in one canonical runtime file and expose aliases here so broad model imports never duplicate definitions or create namespace collisions.

    Reference #

    • Zongyi Li et al., “Fourier Neural Operator for Parametric Partial Differential Equations”, 2020.
