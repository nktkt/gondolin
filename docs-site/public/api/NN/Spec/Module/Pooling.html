    Pooling module wrappers #

    These wrappers expose pooling specs as NNModuleSpecs.

    Conventions:

    • Channel-first images use shape (C, H, W) at the spec level (.dim C (.dim H (.dim W .scalar))).
    • MaxPool2DModuleSpec applies the spatial max-pool independently per channel.
    • AvgPool2DModuleSpec is provided for a single-channel 2D tensor; multi-channel usage typically maps it per channel in the same way as max-pool.

    If you want a PyTorch mapping: nn.MaxPool2d / nn.AvgPool2d on a single (C,H,W) image (no batch).
