    Attention module wrappers #

    This file wraps a few attention blocks as NNModuleSpecs so we can:

    • compose them with SpecChain (shape-safe pipelines), and
    • attach simple export/pretty-print metadata for demos.

    The wrapper below builds a self-attention context with Q=K=V=x and no mask, which matches the common "encoder block" usage. More specialized variants (cross-attention, causal masks, etc.) are defined at the layer-spec level in NN/Spec/Layers/Attention.lean.

    In PyTorch terms, the core computation is scaled dot-product self-attention: softmax(QK^T / sqrt(d)) V, and newer PyTorch exposes it as torch.nn.functional.scaled_dot_product_attention.

    This wrapper stays intentionally narrow: it is self-attention only (Q=K=V=x) with no causal mask.
