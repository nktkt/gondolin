    Scalar #

    Spec-only scalar conventions.

    The spec layer fixes its scalar to ℝ for mathematical reasoning. Runtime scalars remain Float/NeuralFloat and are handled separately.

    References / context:

    • Gondlin paper (overall scalar-polymorphic architecture and trust boundary discussion): arXiv:2602.22631.
    • IEEE 754-2019 is the reference point for the executable Float32 model (IEEE32Exec) used in the runtime/numerics layers (not this file).
