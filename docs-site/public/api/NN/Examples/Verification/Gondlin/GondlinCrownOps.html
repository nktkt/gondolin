    Gondlin CROWN ops #

    Running CROWN end-to-end on small Gondlin graphs.

    We compile Gondlin programs to the verifier IR (NN.IR.Graph), then run:

    • IBP (runIBP)
    • basic CROWN forward bounds (runCROWN)
    • objective-dependent backward/dual CROWN (runCROWNBackwardObjective)

    The goal is to provide compact, fast workflows that exercise the nonlinear ops added to CROWN:

    • softmax (vector)
    • mse_loss (vector → scalar)

    For attention + layer_norm, see NN/Examples/Verification/Gondlin/GondlinTransformerIBP.lean.

    Run: lake exe verify -- gondlin-crown-ops lake exe verify -- gondlin-crown-ops --dtype ieee754exec
