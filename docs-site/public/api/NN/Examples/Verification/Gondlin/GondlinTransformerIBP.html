    Gondlin transformer IBP #

    Small end-to-end workflow:

    Gondlin (MHA + LayerNorm + MSE) → compile to NN.IR.Graph → run:

    • IBP (runIBP)
    • basic CROWN forward bounds (runCROWN)
    • objective-dependent backward/dual CROWN (runCROWNBackwardObjective)

    Run: lake exe verify -- gondlin-transformer-ibp lake exe verify -- gondlin-transformer-ibp --with-crown lake exe verify -- gondlin-transformer-ibp --dtype ieee754exec

                              def NN.Examples.Verification.Gondlin.GondlinTransformerIBP.main (args : List String) :
                              IO Unit

                              CLI entry point for the transformer-IBP workflow.

                              This is wired into lake exe verify -- gondlin-transformer-ibp.

                              By default this command is a fast validation check: compile the Gondlin transformer fragment to the verification IR and run IBP on the scalar loss. Pass --with-crown to also run the experimental transformer-scale CROWN passes. The separate gondlin-crown-ops command keeps CROWN itself in the standard check suite on compact graphs, while this file focuses on the heavier attention/layer-norm front-end path.
