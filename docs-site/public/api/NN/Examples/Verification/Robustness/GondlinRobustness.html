    Gondlin robustness #

    End-to-end robustness certification for a Gondlin model.

    We build a compact 2-class classifier in Gondlin, compile it to the verifier IR, and certify a margin condition on an ℓ∞ input box using:

    • IBP (runIBP)
    • a simple CROWN/affine pass (runAffine + AffineVec.eval_on_box)

    Spec we certify (binary logits): ∀ x ∈ [x0-ε, x0+ε], logit0(x) > logit1(x)

    Run: lake exe verify -- gondlin-robustness lake exe verify -- gondlin-robustness --float32-mode ieee754exec
