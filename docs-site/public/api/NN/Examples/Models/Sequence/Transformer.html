    Transformer Text Example #

    Runnable gondlin transformer example. It reads a local text corpus, builds a byte-level sequence reconstruction sample, and trains one transformer encoder block on that real text window.

    The reusable model wiring lives in NN.API.Models.Transformer (nn.models.transformerEncoder). This file is the runnable wrapper.

    python3 scripts/datasets/download_example_data.py --tiny-shakespeare
lake build -R -K cuda=true && lake exe gondlin transformer --cuda --tiny-shakespeare --steps 1
