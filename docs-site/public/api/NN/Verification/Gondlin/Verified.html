    Verified forward compiler bridge #

    This module is the public naming layer for the verified forward compiler bridge.

    The implementation proof lives in NN.Verification.Gondlin.Proved; this file exposes the names we want users and downstream modules to reach for: compileForward1, compileForward1_wellFormed, and compileForward1_correct.
