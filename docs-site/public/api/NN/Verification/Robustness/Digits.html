    Digits certified robustness #

    Certified accuracy for a linear classifier on sklearn's digits dataset.

    Workflow:

    • Train a small linear classifier on sklearn's digits (8x8) in Python and export:
      • weights JSON (digits_linear_weights.json)
      • test dataset JSON (digits_test.json)
    • In Lean, load the weights + dataset and compute certified accuracy under an ℓ∞ input box using:
      • IBP (runIBP)
      • a simple CROWN/affine pass (runAffine + AffineVec.eval_on_box)

    Run: lake exe verify -- digits lake exe verify -- digits --eps=0.02 --max=360 lake exe verify -- digits --float32-mode ieee754exec --eps=0.02

    Regenerate assets: python3 scripts/verification/robustness/train_digits_linear.py
